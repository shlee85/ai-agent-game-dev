# Wave Plan

## 1) 목적
- `한 판 = 한 Wave` 구조에서 Wave1/2/3의 스폰 규칙, 시간, 전환 흐름을 정의한다.

## 2) 라운드 구조 (확정)
- 플레이어는 한 번 시작할 때 한 개의 Wave만 플레이한다.
- Wave1 클리어 -> Wave2 해금
- Wave2 클리어 -> Wave3 해금
- 각 Wave 종료 후: 결과 화면 -> 상점(아이템 구매) -> 다음 Wave 선택/진입

## 3) Wave 기본 스폰표 (확정)
| Wave | 시간 | 스폰 간격 | 구성 | 총 개체 수 |
|---|---:|---:|---|---:|
| Wave1 | 3분 | 2.2초 | 기본형 100% | 36 |
| Wave2 | 5분 | 1.8초 | 기본형 70% + 돌격형 30% | 90 |
| Wave3 | 7분 | 1.5초 | 모든 적 캐릭터 100% 순환 스폰(현재: 기본형 + 돌격형) | 168 |

## 4) 난이도 연동 규칙 (확정)
- 초보: 각 Wave에서 1종만 출현
- 중수: 각 Wave에서 최대 2종 출현
- 고수: 각 Wave에서 전체 적 캐릭터 출현
- 세부 수치 연동은 `DifficultyPlan.md`를 따른다.

## 5) 종료/패배 규칙
- 승리: 해당 Wave 타이머 종료 또는 스폰 적 전부 처리 + Heart > 0
- 패배: Heart == 0
- 누수 규칙: 적 1기 통과 시 Heart 1 감소

## 6) 테스트 체크리스트
- Wave 해금 순서(W1->W2->W3) 정상 확인
- Wave 종료 후 상점 진입 흐름 정상 확인
- 난이도별 스폰 종류 제한 규칙 정상 확인
