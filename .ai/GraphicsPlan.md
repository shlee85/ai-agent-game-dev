# Graphics Plan

## 1) 목표
- 모바일에서 가독성이 높고, 타워/적/경로/위험도를 즉시 구분 가능한 비주얼을 만든다.
- 베타 단계에서는 "고퀄리티"보다 "명확한 식별성 + 성능 안정성"을 우선한다.
- **최종 아트 스타일: 사이버펑크 / 네온 SF (Cyberpunk / Neon Sci-Fi) 2D 탑다운**

## 2) 아트 방향 (사이버펑크 네온 SF)
- **스타일:** 사이버펑크, 다크 메탈 배경 + 고대비 네온(Neon) 라이팅
- **시점:** 탑다운(Top-down) 2D
- **배경:** 어두운 철판, 기계적인 패널, 그리드 라인
- **렌더링:** 2D 스프라이트 기반 (로봇, 기계, 포탑 위주로 구성하여 형태의 일관성 확보 용이)

## 3) 컬러 시스템 (네온 팔레트)
- **아군(타워):** 시안(Cyan) / 밝은 블루 네온 발광
- **적 유닛(외계 종족/침공 로봇):** 마젠타(Magenta) / 핫 핑크 / 레드 네온 발광
- **스킬(아이템):** 옐로우(Yellow) / 딥 블루(Deep Blue) 등 확실히 대비되는 특수 컬러
- **경로(Path):** 어두운 회색 메탈 바닥 위 흐릿한 파란색 LED 라인
- **위험 상태(하트 저하, 누수):** 붉은색 경고등 점멸 이펙트

## 4) 캐릭터별 세부 디자인 가이드 (프롬프트/에셋 참고용)

### 4.1 방어 캐릭터 (플레이어 측 기계 포탑)
- **Aegis Lance (단일 저격형)**
  - 실루엣: 길고 날렵한 레일건 형태의 포탑
  - 컬러: 어두운 회색 메탈 바닥 + 밝은 시안(Cyan) 라인
  - VFX: 직선으로 뻗어나가는 얇고 강렬한 파란색 레이저 빔
  - 특징: 회전하는 중심 포신, 저격 전 에너지가 모이는 파티클 효과

- **Ember Nova (광역 폭격형)**
  - 실루엣: 둥글고 넓은 반응로(Reactor) 형태
  - 컬러: 건메탈 + 주황/빨간색 네온 플라즈마 코어
  - VFX: 타격 시 둥근 원형의 화염 폭발 링, 파편이 사방으로 튀는 이펙트
  - 특징: 육중한 기계 장치, 포신 대신 중앙 코어에서 에너지가 뿜어져 나옴

- **Frost Ward (감속 제어형)**
  - 실루엣: 육각형 얼음 크리스탈이 박힌 제어탑
  - 컬러: 화이트 메탈 + 밝은 하늘색(Ice Blue) 네온
  - VFX: 주변으로 퍼져나가는 육각형 모양의 빙결 파동 오라
  - 특징: 주변 공기가 얼어붙는 듯한 차가운 안개 파티클

### 4.2 적 캐릭터 (외계 기계 종족)
- **Runner Drone (돌격형)**
  - 실루엣: 바퀴 없이 둥둥 떠다니는 날렵한 쐐기/삼각형 정찰 드론
  - 컬러: 검은색 외관 + 강렬한 마젠타/핑크(Magenta/Pink) 네온 외눈(Eye)
  - VFX: 이동 시 뒤로 남는 붉은색 잔상(Trail) 트레일
  - 특징: 작고 빠름, 기계 곤충 같은 날카로운 움직임

- **Guard Shell (기본/준탱커형)**
  - 실루엣: 두꺼운 장갑판을 여러 겹 두른 육중한 딱정벌레/탱크 형태의 로봇
  - 컬러: 탁한 녹슨 금속 + 진한 빨간색(Deep Red) 네온 코어
  - VFX: 피격 시 장갑에서 튀는 스파크, 이동 시 바닥에 남는 묵직한 이펙트
  - 특징: 움직임이 둔탁하며, 피격될 때마다 장갑판에 빛이 들어옴

## 5) UI/UX 비주얼 가이드
- **HUD:** 투명한 홀로그램 유리판 느낌의 다크 UI 보드
- **아이콘:** `expo-vector-icons`를 활용한 깔끔한 화이트/네온 아이콘 (추후 SF 스타일 PNG 교체)
- **플로팅 텍스트:** 픽셀/디지털 폰트를 활용한 네온 옐로우 데미지/골드 텍스트

## 6) 애니메이션 원칙
- **이펙트 연출:** 과도한 프레임 애니메이션보다는, 스프링 애니메이션(`react-native-reanimated`)과 `Opacity`, `Scale` 변화를 활용한 모던하고 부드러운 스킬 이펙트 지향.
- **피드백:** 적 피격 시 순간적인 흰색 번쩍임(Flash), 타워 발사 시 반동(Recoil) 효과를 통해 타격감 극대화.
