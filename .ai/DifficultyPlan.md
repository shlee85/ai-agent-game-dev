# Difficulty Plan

## 1) 목적
- 난이도(초보/중수/고수)의 수치 정책과 적용 우선순위를 상세 정의한다.
- 본 문서는 `GamePlan.md`의 2단계 확정 기준을 구현 가능한 형태로 분해한다.

## 2) 공통 고정값 (확정)
- 시작 골드: 360
- 플레이어 Heart: 10 (각 Wave 시작 시 초기화)
- 적 이동속도 배율: 1.00x
- 처치 골드 배율: 1.15x

## 3) 난이도별 조정값 (확정)
| 항목 | 초보 | 중수(기본) | 고수 |
|---|---:|---:|---:|
| 적 체력 배율 | 0.85x | 1.00x | 1.20x |
| 적 출현량 배율 | 0.90x | 1.00x | 1.20x |
| 적 출현 캐릭터 종류 수 | 1종 | 2종 | 전체 종 |
| 다이아 보너스(클리어) | +10% | 기준값 | -10% |

## 4) 적용 우선순위
1. Wave 기본 스폰표 로드
2. 난이도별 출현 캐릭터 종류 수 적용
3. 난이도별 출현량 배율 적용
4. 적 체력 배율 적용
5. 보상(다이아) 배율 적용

## 5) 구현 체크포인트
- 난이도 변경은 로비에서만 허용
- 인게임 진입 후 난이도 변경 잠금
- 각 Wave 시작 시 Heart를 10으로 강제 초기화

## 6) 테스트 체크리스트
- 난이도 변경 후 첫 Wave에서 수치 즉시 반영 확인
- 초보/중수/고수에서 적 수와 체력 체감 차이 확인
- 10회 반복 플레이 시 난이도별 클리어율 차이 확인
