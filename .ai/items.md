# Item Design (Beta)

## 1) 문서 목적
- 본 문서는 다이아 재화를 사용하는 전투 보조 아이템 설계를 정의한다.
- 핵심 게임 룰 및 완료 기준은 `GamePlan.md`에서 관리한다.

## 2) 재화 구조
- 골드(Gold): 전투 중 타워 설치/업그레이드에 사용
- 다이아(Diamond): 전투 전/중 아이템 구매 및 사용에 사용

## 3) 베타 아이템 목록 (초기 3종)

### 3.1 EMP Pulse
- 종류: 즉시 발동 / 광역 제어
- 효과: 맵 내 모든 적 2.5초 정지
- 가격: 30 다이아
- 쿨다운: 없음(소모형)
- 사용 제한: 판당 1회
- 용도: 누수 직전 긴급 정지

### 3.2 Orbital Strike
- 종류: 즉시 발동 / 광역 공격
- 효과: 선택 지점 반경 내 고정 피해 120
- 가격: 45 다이아
- 쿨다운: 없음(소모형)
- 사용 제한: 판당 1회
- 용도: 웨이브 밀집 구간 정리

### 3.3 Overclock Core
- 종류: 버프 / 시간제
- 효과: 8초 동안 아군 타워 공격속도 +35%
- 가격: 35 다이아
- 쿨다운: 없음(소모형)
- 사용 제한: 판당 1회
- 용도: 웨이브 피크 타이밍 화력 증폭

## 4) 사용 규칙
- 아이템은 전투 중 HUD에서 즉시 사용 가능해야 한다.
- 각 아이템은 해당 판에서 1회만 사용 가능하다(베타 정책).
- 소지 다이아가 부족하면 구매/사용이 불가능하다.
- 배속 x2 상태에서도 아이템 지속시간은 실시간 기준으로 동일하게 유지한다.

## 5) UI/UX 규칙
- HUD에 아이템 슬롯 3개를 고정 표시한다.
- 각 슬롯에 아이콘, 가격, 사용 가능 여부(활성/비활성)를 표시한다.
- 사용 직후 시각 효과와 로그 텍스트를 표기한다.

## 6) 밸런스 원칙
- 아이템은 "위기 탈출 보조" 역할로 제한한다.
- 아이템 1개만으로 웨이브 승패가 확정되지 않도록 피해량/지속시간을 조절한다.
- 타워 조합 전략이 핵심이고, 아이템은 보조 수단이어야 한다.

## 7) 테스트 체크리스트
- 다이아 부족 시 구매/사용 차단 동작 확인
- 판당 1회 제한 동작 확인
- 배속 x1/x2에서 아이템 효과 시간 동일성 확인
- 다중 아이템 연속 사용 시 크래시/프리징 없음 확인
