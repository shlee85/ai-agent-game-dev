# Dev Setup

## 1) 필수 설치 (Ubuntu)
```bash
sudo apt update
sudo apt install -y nodejs npm
node -v
npm -v
```

## 2) Expo 의존성 설치
```bash
npm install
```

## 3) 실행
```bash
npm run start
```

## 4) 현재 생성된 기본 구조
- `App.tsx`
- `src/data/difficulty.ts`
- `src/data/waves.ts`
- `src/screens/WaveScreen.tsx`
- `src/ui/HUD.tsx`
