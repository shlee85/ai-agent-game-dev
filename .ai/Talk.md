-> 터미널에서 아래 명령어로 바로 세팅할 수 있어:
```bash
git init
git add .
git commit -m "feat: 반응형 그리드 맵 및 기본 환경 세팅"
git branch -M main
```
-> 깃허브에 레포지토리 만들고 나서 원격(remote) 연결만 추가하면 끝이야. 어떻게 할지 결정되면 알려줘! 내가 커밋까지 도와줄게.
